# clertonraf.github.io

Personal website and CV for [Clerton Araujo](https://www.clerton.dev), Senior Full Stack Engineer based in Berlin.

## Structure

```
.
├── index.html          # English version
├── de/index.html       # German version
├── pt-br/index.html    # Portuguese (BR) version
├── style.css           # Shared stylesheet
├── sitemap.xml         # Multilingual sitemap (hreflang: en / de / pt-BR)
├── robots.txt
├── llms.txt            # AI-readable profile (llms.txt convention)
├── CNAME               # Custom domain: www.clerton.dev
├── Clerton_Araujo_Senior_FullStack.pdf
├── Clerton_Araujo_Senior_FullStack.docx
├── Clerton_Araujo_Lebenslauf_DE.pdf
├── Clerton_Araujo_Lebenslauf_DE.docx
├── Clerton_Araujo_Curriculo_PT.pdf
└── Clerton_Araujo_Curriculo_PT.docx
```

## CV downloads

Each language version of the site offers two formats:

| Language | PDF | DOCX (ATS) |
|---|---|---|
| English | `Clerton_Araujo_Senior_FullStack.pdf` | `Clerton_Araujo_Senior_FullStack.docx` |
| German | `Clerton_Araujo_Lebenslauf_DE.pdf` | `Clerton_Araujo_Lebenslauf_DE.docx` |
| Portuguese | `Clerton_Araujo_Curriculo_PT.pdf` | `Clerton_Araujo_Curriculo_PT.docx` |

PDF is the primary download. DOCX is provided as a secondary option for recruiters who need to upload to ATS (Applicant Tracking Systems) that parse plain text.

## Tech

- Static HTML + CSS — no build step, no framework
- Hosted on GitHub Pages with a custom domain (`www.clerton.dev`)
- Dark terminal aesthetic, monospace font, responsive layout
- Lighthouse score: 100 across all categories (Performance, Accessibility, Best Practices, SEO)
- Structured data: JSON-LD Person schema, Open Graph, Twitter Card, hreflang
- `llms.txt` for AI crawler discoverability

## Updating the CV files

The `.docx` files are generated by `generate_cv.py` (not committed) using `python-docx`. The `.pdf` files are generated by `generate_pdf.py` (not committed) using `weasyprint`.

To regenerate after content changes, run each script locally and commit the output files.
